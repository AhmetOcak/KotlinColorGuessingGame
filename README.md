# KotlinColorGuessingGame

Tahmin doğruysa "CORRRECT :)" yazısı çıkar.

If the guess is correct, the text "CORRRECT :)" is displayed.

</br>

Tahmin yanlış ise "INCORRECT :(" yazısı çıkar.

If the guess is wrong, "INCORRECT :(" is displayed.

</br>
